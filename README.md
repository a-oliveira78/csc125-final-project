# CSC-125-201 Final Project

## Project Description
My project is a bookkeeping record system that creates an account for every user, logs every book that they take out, and prevents them from buying another book until they return at least 1.

## Features
- Dictionary storing Account name (key) and books (value)
- Custom function create_account() that Adds a user account, with an empty list
- for loop that Shows the menu until the user says 'no'
- User input with validation
- Formatted output
